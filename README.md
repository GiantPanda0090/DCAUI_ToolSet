# DCAUI_ToolSet

Practice toolset for Cisco Automating Cisco Data Center Solutions. Inspired by Cisco DCAUTI Training  

Testbed for Elixir Data Center Network Automation  

Module list:  
aci - ACI Automation  
nxos -  NXOS Automation  
ucs - UCS Automation  

Parameter for Login:  
param - Parameter for all module
